### 🚧 进行中的项目

- **PINN 与谱方法的深度融合研究** 探索将 PINN 与高精度谱方法（如傅里叶、切比雪夫等）结合，利用频域正交基提升空间导数精度与网络收敛速度，以增强复杂物理问题的求解能力。

- **基于 PINN 的 Navier-Stokes 方程建模与预测** 应用 PINN 求解不可压缩流体 N-S 方程。引入自适应非均匀采样（如 Sobol 序列）与物理先验约束，减少样本浪费并缓解训练震荡，旨在开发适用于湍流及边界层分析的泛化模型。

- **多尺度可解释物理神经网络开发** 构建多尺度嵌套网络结合不确定性估计，增强复杂动力系统建模能力。计划融合图神经网络（GNN）处理非规则网格数据，并显式引入边界与守恒约束提升物理可解释性。

- **科研笔记与教学资源开源分享** 梳理 PINN、数值方法与深度学习的交叉研究（涵盖经典方程求解、训练技巧与误差分析），形成系统化学习资源库，将以博客或 Wiki 形式在 FEMATHS 官网持续开源更新。
